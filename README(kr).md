# Lemon guild

Lemon guild는 마인크래프트 베드락 에디션의 애드온입니다.
이 애드온은 마인크래프트에 길드 기능을 추가합니다.

## 적용법

릴리스에서 최신 릴리스를 다운로드하세요.
다운로드한 후 Minecraft에 로드하세요.
월드 설정에서 실험적인 홀 기능과 베타 API 기능을 활성화하세요.

## 사용법

크리에이티브 인벤토리 하단에 있는 ["길드"](./items/guild_ui_open_item.json) 아이템을 꺼냅니다.
(또는 /give @s Lemon:guild_ui_open_item으로 꺼낼 수 있습니다)
["길드"](./items/guild_ui_open_item.json) 아이템을 사용하면 사용자 인터페이스가 나타납니다.
